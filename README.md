# Proyecto Urban Grocers

## Pruebas al back-end a aplicación de Plataforma de entrega de productos por catálogo y kits preparados

**Logros en el proyecto Urban Grocers:**

- Elaboré una **lista de comprobación (checklist) para validar una nueva funcionalidad del backend de la API**, asegurando que todos los endpoints cumplieran con los requisitos funcionales y de negocio.
- Realicé pruebas **manuales de API utilizando Postman**, enviando solicitudes `GET`, `POST`, `PUT` y `DELETE` con distintos cuerpos de datos (válidos e inválidos) para verificar:
    - Respuestas del servidor (códigos de estado HTTP)
    - Mensajes de error apropiados
    - Validación de campos requeridos
    - Integridad de datos en respuestas JSON
- Detecté y documenté **bugs y comportamientos inesperados** directamente en **Jira**, con descripciones detalladas, pasos para reproducirlos y evidencias adjuntas (capturas de pantalla o exportaciones de Postman).
- Verifiqué aspectos clave como:
    - Autenticación de usuarios
    - Accesos no autorizados
    - Manejo de productos en inventario desde la API
    - Lógica de actualización de datos (por ejemplo, stock y precios)
- Participé en el flujo de trabajo tipo Scrum.


### Herramientas usadas: Postman, JIRA
